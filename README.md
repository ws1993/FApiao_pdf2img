# PDF 转图片工具

一个可直接部署在 Vercel 的批量 PDF 转 JPG 工具，支持：

- 批量上传 PDF。
- 逐文件转换进度展示。
- 单个 PDF 单独下载（zip）。
- 全部结果打包下载（zip）。
- 转换过程在浏览器本地完成，不上传文件到服务器。

## 本地预览

这是纯静态页面，无需安装依赖。可任选以下方式启动本地静态服务：

```bash
python -m http.server 5173
```

然后访问 `http://localhost:5173`。

## Vercel 部署

仓库已包含 `vercel.json`，直接连接仓库即可部署，无需额外构建步骤。

## 测试文件

示例 PDF 放在项目目录下的 `pdf/` 文件夹，可直接用于功能验证。
